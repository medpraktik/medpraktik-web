Aktivasi MedPraktik

Download installer, kirim fingerprint, lalu tunggu license key.

Halaman ini dipakai setelah pembayaran atau instruksi dari tim MedPraktik. Simpan kode aktivasi Anda karena kode ini dipakai untuk melihat progress dan license key.

Kode aktivasi Memuat...

1. Download installer resmi

Installer hanya bisa digunakan dengan license key resmi. Pembayaran tetap diverifikasi admin sebelum license dikirim.

Jika download gagal, kirim kode aktivasi ke WhatsApp MedPraktik.

2. Ambil fingerprint perangkat

  1. Install MedPraktik di laptop Windows yang akan dipakai praktik.
  2. Buka MedPraktik.
  3. Masuk ke layar Aktivasi/Lisensi.
  4. Copy fingerprint perangkat dari layar tersebut.
  5. Paste fingerprint di form aktivasi di bawah.

3. Kirim data aktivasi

Kirim WhatsApp
